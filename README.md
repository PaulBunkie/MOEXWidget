# MOEX Widget - Android виджет котировок MOEX

Android-приложение с домашним виджетом, который отображает график цены выбленной акции за последние 24 часа и обновляется автоматически раз в час.

## Функциональность

- **Виджет на рабочем столе** - отображает график цены акции за 24 часа
- **Автоматическое обновление** - раз в час через WorkManager
- **Ручное обновление** - тап на виджет для немедленного обновления
- **Кэширование** - локальное хранение последнего результата
- **Fallback** - при отсутствии интернета использует кэш

## Технологии

- **Kotlin** - основной язык программирования
- **Android AppWidget API** - для создания виджета
- **WorkManager** - для фоновой синхронизации
- **OkHttp** - HTTP клиент для запросов к MOEX ISS API
- **Gson** - парсинг JSON ответов
- **Canvas** - рисование графика как Bitmap

## Установка

1. Откройте проект в Android Studio
2. Подождите пока Gradle синхронизирует зависимости
3. Запустите приложение на устройстве или эмуляторе
4. Добавьте виджет на рабочий стол

## Использование

1. Добавьте виджет "MOEX Stock Chart" на рабочий стол
2. Виджет автоматически загрузит данные S SBER за последние 24 часа
3. Для ручного обновления нажмите на виджет
4. Виджет обновляется автоматически раз в час

## API

Приложение использует Moscow Exchange ISS API:
- Endpoint: `https://iss.moex.com/iss/engines/stock/markets/shares/boards/TQBR/securities/{ticker}/candles.json`
- Интервал: 60 минут
- Период: последние 24 часа

## Расширения (необязательно)

- Выбор тикера пользователем
- Несколько акций в списке
- Переключение интервалов (1h / 5m / 1d)
- Кэш истории за неделю
- Уведомления при резком изменении цены
